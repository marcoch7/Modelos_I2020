Dependencias

    -Para correr el programa es necesario tener instalado python3 
    -Es necesario tener dentro de la carpeta del programa un documento llamado lote.csv


Compilar y correr

    -Para compilar y correr el programa, cambie la extension del archivo "B61868.txt" a "B61868.py" y escriba en la terminal: make build
    -O bien, despues de cambiar la extension del archivo, escriba en la terminal python3 B61868.py


Descripcion del programa

    -Este programa lee el documento "lote.csv", guarda los parametros A, B, C y D como variables
    y calcula las probabilidades de cada caracteristica. A partir de estos resultados calcula tambien las 
    probabilidades de que ocurran estas caracteristicas dadas otras asi como las dependencias o independencias 
    que hay entre estas.
